# 메시지 중복 문제 테스트 계획

## 문제 정의
- 채널에서 메시지를 보냈을 때 2번 표시됨
- 채널에서 나갔다가 들어오면 메시지가 2번씩 표시됨

## 테스트 환경
- URL: http://localhost
- 로그인: testuser2@test.com / 123456

## 테스트 케이스

### TC1: 메시지 전송 후 1번만 표시되는지 확인
1. 로그인 후 /chat 페이지로 이동
2. 팀 선택 → 채널 선택
3. 메시지 입력 후 전송
4. 화면에 메시지가 1번만 표시되는지 확인
5. 브라우저 콘솔(F12)에서 에러 확인

### TC2: 채널 재진입 시 중복 확인
1. TC1 완료 후
2. 다른 채널로 이동
3. 다시 원래 채널로 이동
4. 메시지가 1번만 표시되는지 확인
5. 메시지 개수 확인 (중복 없음)

### TC3: 새로고침 후 메시지 확인
1. 페이지 새로고침 (F5)
2. 채널 선택
3. 메시지가 1번만 표시되는지 확인
